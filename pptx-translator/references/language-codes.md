## Engine Differences

### Amazon Translate
- Better for: High volume, cost-sensitive, straightforward content
- Supports custom terminology (CSV upload)
- More consistent with technical documentation

### Bedrock LLM
- Better for: Nuanced content, marketing copy, context-sensitive translation
- Supports glossary for term consistency
- Can adapt style (formal, casual, technical)
- Better at preserving tone and intent
